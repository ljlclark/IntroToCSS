The ID Selector

  <head>
    <title>The Style Element</title>
    <meta charset="utf-8" />
    <style>
      #title {
        background: lightsteelblue;
      }
    </style>
  </head>

An ID can only be assigned to one element in a document.

An ID selector defines the single element to which the style properties are applied.

Using the above "#title" rule example; only the element that is qualified with the attribute (id="title") will use the #title ID style properties.
